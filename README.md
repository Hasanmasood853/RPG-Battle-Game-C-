# RPG-Battle-Game-C-
A simple console-based RPG battle game built using Object-Oriented Programming (OOP) concepts in C++
The project demonstrates:

Inheritance
Polymorphism
Function overriding
Encapsulation
Virtual functions
Basic game logic
Features
Two playable characters:
Warrior
Mage
Turn-based combat system
Warrior special abilities:
50% extra attack damage
Armor defense system
Mage special abilities:
Mana-based magic attacks
Double damage spell attacks
Health and alive status tracking
Input validation
Round-by-round battle statistics
